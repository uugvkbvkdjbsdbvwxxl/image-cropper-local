# 本地图片裁剪

一个纯前端图片裁剪工具，图片只在浏览器本地处理，不上传到服务器。适合快速裁剪头像、封面图、社交媒体配图和固定尺寸素材。

## 在线访问

- GitHub Pages：https://uugvkbvkdjbsdbvwxxl.github.io/image-cropper-local/
- Netlify：https://image-cropper-local-20260707-wxl.netlify.app/

## 功能

- 本地导入图片，支持拖拽上传
- 拖动裁剪框，移动图片视图，滚轮缩放
- 常用比例裁剪：自由、1:1、4:3、16:9、3:4、9:16
- 常用尺寸预设：公众号封面、头像、竖版海报、横版封面等
- 自定义输出宽高，并可锁定比例联动输入
- 旋转、重置、适合屏幕
- 撤销和重做操作
- 实时预览输出尺寸
- 支持 PNG、JPEG、WebP 导出
- 支持导出倍数和 JPEG/WebP 质量设置
- 可直接复制裁剪后的图片到剪贴板
- 支持系统明暗模式

## 使用

直接打开 `index.html` 即可使用，不需要安装依赖。

也可以用任意静态服务器预览：

```bash
npx serve .
```

## 隐私

应用不包含后端服务，图片处理全部发生在浏览器内。选择的图片不会上传、保存或发送到远程服务器。

## 部署

这是一个静态网页，可以部署到 GitHub Pages、Netlify、Vercel 或任何静态托管服务。
